# Estudos Aprofundados em Python para Data Science

Este repositório documenta a minha jornada de consolidação técnica em Python e Estatística, visando a preparação para Machine Learning avançado.

**Objetivo:** Transcender a sintaxe básica e aplicar "Best Practices" de engenharia de software na análise de dados.

## Estrutura do Projeto
* **01_fundamentos:** Tipos primitivos, controle de fluxo e tratamento de exceções.
* **02_estruturas_dados:** Manipulação avançada de Listas e Dicionários para limpeza de dados.

## Ferramentas
* Python 3.x
* Jupyter Notebook
* Git/GitHub
